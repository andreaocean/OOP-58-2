++++++++++ Команда на один раз для проекта +++++++++++
git init      | инициализация репозитория
git remote add origin https://github.com/andreaocean/OOP-58-2.git


++++++++++ Команды, которые часто используются ++++++++++

git status   | статус проекта
git add .     | добавить все изменения в буфер обмена
git commit -m "message" | сохранить изменения, которые в буфере обмена
git push origin master | main    | отправить сохраненные изменения в репозиторий
